## Getting Started

Follow these steps to run the project locally:

### 1. Clone the repository

```bash
git clone https://github.com/Qubetronics/eventverseimam.git
cd eventverseimam
````

### 2. Install dependencies

```bash
npm install
```

### 3. Run the project

* For development mode:

```bash
npm run dev
```

* For production mode:

```bash
npm run build
npm start
```
