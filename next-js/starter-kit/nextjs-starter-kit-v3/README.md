# Next JS Starter Kit

A robust starter kit built with Next.js for professional developers. This kit is designed to accelerate development by integrating modern tools and best practices out of the box.

## Live Project - Vercel 
  [Next JS Starter Kit](https://safi-nextjs-starter-kit.vercel.app/)

## Themeforest Source Code

## Features

- **Next.js 15**: Leverage the latest features of Next.js for a modern and performant app.
- **React 19**: Utilize React 19's cutting-edge capabilities for seamless user interfaces.
- **Tailwind CSS**: Rapidly build beautiful and responsive designs.
- **Prisma ORM**: Effortlessly manage your PostgreSQL database with Prisma.
- **PostgreSQL**: A reliable and scalable relational database.
- **Authentication**: Secure and customizable authentication flows included.

## Installation

1. Clone the repository:
  ```bash
    git clone https://github.com/safi-siddiqui-github/nextjs-starter-kit.git
    cd next-js-starter-kit
  ```

2. Zip the Project Files:
  If purchased from Themeforest

3. Install node_modules:
  First, install npm packages:
  ```bash
    npm install
  ```

4. Create .env.local file:
  Create an env file and fill the following Variables:
  ```bash
    PRISMA_POSTGRES_DATABASE_URL=
    PRISMA_POSTGRES_ACCELERATE_DATABASE_URL=
    PRISMA_PULSE_API_KEY=
    CUSTOM_SESSION_SECRET_KEY=
  ```

5. Run the project:
  Run your project locally:
  ```bash
    npm run dev
  ```
